# Quick Start

## Outcome

load LightRAG architecture context, retrieval boundaries, graph-RAG setup notes, and acceptance checks into your AI host before building a RAG workflow

## Two-Minute Path

1. Read `AGENTS.md`.
2. Copy the relevant host instruction into your AI coding host.
3. Run `06_EVALS/smoke_check.md`.
4. If the check fails, open `03_PITFALL_LOG.md`.
